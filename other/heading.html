What is Java Constructor?

A Constructor in Java is a special method that is used to intialize objects.It contains collection of statements(i.e. instructions) that are excecuted at a time of object creation. Every class has a constructor which is called when an object is created in class.

If we do not explcitly write a constructor for class, the Java compiler builds a default constructor for that class. Each time a new object is created, at least one constructor will be invoked. The main rule of constructors is that they should have the same name as the class. A class can have more than one constructor. Constructors have only one purpose i.e. Creating an object for the class and initializing the instance variables.
